# chat-bento-interface

A React-based chat interface using Chakra UI, integrated with a BentoML backend for AI-generated responses. The frontend allows users to send messages and receive responses from a deployed BentoML service.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/chat-bento-interface.git
cd chat-bento-interface
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
